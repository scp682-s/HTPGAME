下面是一份**可直接交给 Claude Code 的 AI 话术模板**，用于生成“房树人拼图游戏”心理分析报告。Claude Code 可以根据这份模板，在你的 Node.js 后端（Vercel Serverless Function）中集成调用大模型的逻辑。

---

## 📋 话术模板（Prompt Template）

```text
你是一位资深房树人心理投射分析专家，用户刚刚完成了一款“房树人拼图游戏”。请根据以下游戏数据，写一段不超过300字的个性化心理分析报告。

【游戏数据】
- 图片主题：{{image_name}}
- 拼图难度：{{grid_size}} x {{grid_size}}
- 完成用时：{{minutes}}分{{seconds}}秒
- 总步数：{{moves}}
- 特殊模组开启情况：{{modifiers}}
- 行为特征：{{behavior_traits}}

【房树人心理分析理论要点】
- 房子：代表家庭安全感、对家庭的态度。拼图中房子部分放置准确、步数少 → 家庭安全感强；反复调整 → 可能对家庭角色有思考或不确定。
- 树：代表生命力、成长经历、自我发展。树干稳固、树冠茂盛 → 生命力旺盛；若拼图时树的部分犹豫 → 可能在成长中经历过挑战。
- 人：代表自我认知、人际关系。人物拼图果断 → 自我接纳度高；反复修正 → 可能对自我形象有较高要求或正在探索。

【分析要求】
1. 语气温暖、鼓励，避免生硬说教。
2. 结合用户的具体数据（例如用时短/步数少 → 高效、果断；用时长/步数多 → 细致、周密）。
3. 给出1-2条贴合用户表现的成长建议（与房树人意象相关）。
4. 整体字数控制在300字以内。

【输出格式】
直接输出分析正文，不要加额外说明或标题。
```

## 🧩 给 Claude Code 的集成说明（可直接复制）

> **任务**：将上述 prompt 模板集成到现有的 Node.js + Vercel 后端 API（`api/generate_report.js`）中，替换原有的简单提示词。  
> **要求**：  
> 1. 从请求体（`req.body`）中提取 `time_seconds`, `moves`, `grid_size`, `modifiers`, `image_name`。  
> 2. 计算 `minutes`、`seconds`，并根据数据生成 `behavior_traits` 字符串（如“高效、果断”或“细致、周密”）。  
> 3. 将 `modifiers` 对象格式化为可读字符串（例如“开启了旋转模组、隐藏碎片”）。  
> 4. 使用模板字符串填充上述模板，然后调用 OpenAI SDK（或兼容的 DeepSeek 等）发送给大模型。  
> 5. 返回 JSON `{ success: true, report: 模型输出 }`，并处理错误。  
> 6. 确保环境变量 `OPENAI_API_KEY` 已配置，若使用其他模型需修改 `baseURL` 和 `model`。

## 🔧 参考实现代码片段（Node.js）

Claude Code 可以根据以下片段自动生成完整代码：

```javascript
// 在 api/generate_report.js 中
const behaviorTraits = (time, moves, size) => {
  if (moves < size * size * 1.5) return '高效、果断';
  if (moves > size * size * 3) return '谨慎、周密思考';
  return '稳健、有条理';
};

const modifiersText = (mod) => {
  const active = [];
  if (mod.rotation) active.push('旋转拼图');
  if (mod.hidden) active.push('隐藏碎片');
  if (mod.trickster) active.push('捣蛋鬼干扰');
  return active.length ? active.join('、') : '无特殊模组';
};

const prompt = `
你是一位资深房树人心理投射分析专家，用户刚刚完成了一款“房树人拼图游戏”。请根据以下游戏数据，写一段不超过300字的个性化心理分析报告。

【游戏数据】
- 图片主题：${image_name}
- 拼图难度：${grid_size} x ${grid_size}
- 完成用时：${minutes}分${seconds}秒
- 总步数：${moves}
- 特殊模组开启情况：${modifiersText(modifiers)}
- 行为特征：${behaviorTraits(moves, time_seconds, grid_size)}

【房树人心理分析理论要点】
...（同上，可以简写或保留完整）
...
`;

// 调用 AI 并返回
```

## ✅ 使用建议

- 将以上话术模板和集成说明 **直接发送给 Claude Code**（或复制到对话中），它会自动生成可运行的 Node.js 代码。  
- 如果担心 token 消耗，可以把模板中的理论要点精简为 2-3 句核心提示。  
- 部署后测试：完成一局游戏，点击“查看心理分析报告”，观察返回的内容是否符合预期。  

这样你无需手动写复杂 prompt，直接让 Claude Code 完成集成工作。
