# 手动安装步骤（推荐）

如果自动安装脚本出现问题，请按照以下步骤手动安装：

## 步骤1：安装Python依赖

打开命令提示符（按 Win+R，输入 cmd，回车），然后执行：

```bash
cd "C:\Users\redking\Desktop\大创项目\心宇宙重塑：房树人图像趣拼\backend"
pip install flask flask-cors openai python-dotenv
```

或者使用 requirements.txt：

```bash
pip install -r requirements.txt
```

如果速度慢，可以使用国内镜像：

```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

## 步骤2：创建配置文件

在 `backend` 目录下，复制 `.env.example` 文件并重命名为 `.env`

Windows命令：
```bash
cd backend
copy .env.example .env
```

## 步骤3：获取API密钥

1. 访问 https://platform.deepseek.com/
2. 点击右上角"注册"
3. 使用手机号注册（支持中国大陆手机号）
4. 登录后，点击左侧"API Keys"
5. 点击"创建新的API Key"
6. 复制生成的密钥（格式：sk-xxxxxxxxxxxxxxxx）

## 步骤4：配置API密钥

用记事本打开 `backend\.env` 文件，修改为：

```
DEEPSEEK_API_KEY=sk-你复制的密钥
```

**注意**：
- 等号两边不要有空格
- 不要加引号
- 保存后关闭文件

## 步骤5：启动后端服务

在命令提示符中执行：

```bash
cd backend
python app.py
```

看到以下信息表示成功：
```
* Running on http://0.0.0.0:5000
* Running on http://127.0.0.1:5000
```

**保持这个窗口打开！** 不要关闭，后端服务需要一直运行。

## 步骤6：集成前端

用文本编辑器打开 `index.html` 文件，在 `</body>` 标签之前添加：

```html
<!-- 心理分析报告功能 -->
<script src="frontend-integration.js"></script>
```

保存文件。

## 步骤7：测试功能

打开新的命令提示符窗口，执行：

```bash
cd "C:\Users\redking\Desktop\大创项目\心宇宙重塑：房树人图像趣拼\backend"
python test_api.py
```

如果所有测试通过，说明配置成功！

## 步骤8：开始使用

1. 确保后端服务正在运行（步骤5的窗口保持打开）
2. 用浏览器打开 `index.html` 文件
3. 开始游戏，选择游戏提供的4张图片之一
4. 完成拼图后，点击"查看心理分析报告"按钮

## 常见问题解决

### 问题1：提示"python不是内部或外部命令"

**解决方法**：
1. 下载Python：https://www.python.org/downloads/
2. 安装时**务必勾选** "Add Python to PATH"
3. 安装完成后重启命令提示符

### 问题2：pip install 失败

**解决方法**：
```bash
# 升级pip
python -m pip install --upgrade pip

# 使用国内镜像
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 问题3：后端启动后提示"Address already in use"

**解决方法**：端口5000被占用，修改端口

编辑 `backend\app.py`，将最后一行改为：
```python
app.run(host='0.0.0.0', port=5001, debug=True)
```

同时编辑 `frontend-integration.js`，将第一行改为：
```javascript
const API_BASE_URL = 'http://localhost:5001';
```

### 问题4：生成报告时提示"API Key错误"

**解决方法**：
1. 检查 `backend\.env` 文件中的密钥是否正确
2. 确保格式为：`DEEPSEEK_API_KEY=sk-你的密钥`
3. 没有多余的空格或引号
4. 保存后重启后端服务

### 问题5：提示"不支持分析用户上传的图片"

**这是正常的！** 系统设计就是这样：
- ✅ 只能分析游戏提供的4张标准房树人图像
- ❌ 不能分析用户上传的自定义图片
- 这是为了保护用户隐私和确保分析准确性

**解决方法**：选择游戏内置的4张图片之一即可。
